# Huazo Web - Clothes Rental Informational Website

An Angular-based informational website for the Huazo clothes rental application.

## Project Setup

This project was created with Angular 18 and uses standalone components with routing.

## Development

### Prerequisites
- Node.js (v20.19.0 or higher recommended)
- npm (v8.0.0 or higher)

### Getting Started

1. Install dependencies (already done):
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```
   The app will be available at `http://localhost:4200/`

3. Build for production:
   ```bash
   npm run build
   ```
